# Urdu Flipbook Platform - Assets

## Audio Files

Place your page-flip sound file here as `page-flip.mp3`.

### Instructions:
1. Find a soft, paper-like page-flip sound (short, ~0.5-1 second)
2. Rename it to `page-flip.mp3`
3. Place it in this `/public` folder

### Alternative Sources:
- Search for "page flip sound effect" or "book turning sound"
- Ensure it's a soft, natural paper sound
- Format: MP3, mono or stereo, short duration

## Fonts

We're using Google Fonts for Noto Nastaliq Urdu (loaded via the web).

If you prefer to use local fonts:
1. Download Jameel Noori Nastaleeq or Noto Nastaliq Urdu
2. Create a `/public/fonts` folder
3. Place font files there (e.g., NotoNastaliqUrdu-Regular.woff2)
4. Update the font loading code in `src/app/layout.tsx`
